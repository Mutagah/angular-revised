Reactive Forms

  • Logic is in the component
  • They are more robust, usable, scalable and testable
  • Suitable for complex forms

We define the structure of the form in the component class

To bind the form, we use formGroup directive and ensure we import the FormsModule in the import array on the module

To bind controls in a reactive form we use formControlName

Form Validations in reactive forms is achieved through giving the second argurment in the FormContol method in your typescript file.

This is how we add custom error messages and we get access to the controls using myForm.get() method
